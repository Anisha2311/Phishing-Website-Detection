# Phishing Website Detection

## Project Overview

A Machine Learning project designed to detect phishing websites based on URL and website-related features.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Workflow

1. Data Preprocessing
2. Feature Engineering
3. Exploratory Data Analysis
4. Model Building
5. Evaluation

## Models Used

- Logistic Regression
- Random Forest
- Decision Tree
