# Phishing Website Detection

## Project Overview

This project focuses on detecting phishing websites using Machine Learning techniques. The objective is to classify websites as legitimate or phishing based on various URL and website-related features. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation.

## Objectives

* Identify phishing websites using machine learning models.
* Perform data preprocessing and feature engineering.
* Analyze website characteristics through exploratory data analysis.
* Compare multiple classification algorithms.
* Evaluate model performance using standard metrics.

## Technologies Used

* Scikit-Learn

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Prediction and Analysis

## Models Used

* Logistic Regression
* Random Forest Classifier
* Decision Tree Classifier

## Features of the Project

* Data Cleaning and Transformation
* Feature Engineering
* Data Visualization
* Model Comparison
* Performance Evaluation

## Results

The models were trained and evaluated to classify phishing and legitimate websites. Performance was assessed using classification metrics and accuracy scores to identify the most effective model.
